How to update

Before updating make sure your Angular packages (@angular/...) are up to date. For example if you would like to update from version {{ libraryPredecessorMajorVersion }}.x.x to version {{ libraryMajorVersion }}.x.x of our SBB Angular packages (@sbb-esta/...), you would first need to update the Angular packages to version {{ libraryMajorVersion }}.x.x. See update.angular.io for a step-by-step guide for updating Angular. Don't forget to update @angular/cdk as well.

To update the SBB Angular packages, use the ng update command. ng update will update all related packages and will automatically run update schematics which helps keeping up with breaking changes. 

ng update @angular/core @angular/cli @angular/cdk @sbb-esta/angular-core
