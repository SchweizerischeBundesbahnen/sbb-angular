SBB Angular Component Library

Our customers should find their way around SBB's digital channels and get to the information they need as quickly as possible. This only works with a consistent user experience across all channels, guaranteeing a uniform appearance across all platforms.

SBB's digital guidelines form the visual guidelines in the new media and are binding on both internal and external projects. They provide clarity on the most important and everyday brand-related questions in online media.

This library is being developed on GitHub (sbb-design-systems/sbb-angular). Any contribution is always welcome. If you would like to report a problem or have a feature request, feel free to open an issue.

@sbb-esta/angular-public vs @sbb-esta/angular-business

We provide two different packages @sbb-esta/angular-public and @sbb-esta/angular-business for two different purposes.

@sbb-esta/angular-public is intended for public websites (e.g. sbb.ch or similar). It uses a large amount of whitespace and supports responsive design for mobile, tablet, desktop, 4k and 5k. The components are more limited in their usage to adhere to the design specification at digital.sbb.ch Design System Web.

@sbb-esta/angular-business is intended for business applications. It uses a compact design and supports responsive design for mobile, tablet and desktop. The components are more flexible in their usage as specified at digital.sbb.ch Design System Business Applications.

If you are uncertain which library to use, we recommend you start with @sbb-esta/angular-business.

  • @sbb-esta/angular-public Components which can be used for public websites.
  • @sbb-esta/angular-business Components which can be used for business apps.
  • @sbb-esta/angular-core Common functionality for @sbb-esta/angular-public and @sbb-esta/angular-business.
  • @sbb-esta/angular-icons Icons provided as components.
  • @sbb-esta/angular-keycloak Authentication module for use with keycloak (sso.sbb.ch).
  • @sbb-esta/angular-maps Components to display 2D/3D maps and to provide map interaction.
