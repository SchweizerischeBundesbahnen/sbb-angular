API reference for Angular SACL textexpand

import { TextexpandModule } from '@sbb-esta/angular-public';

Directives

TextexpandComponent

Selector: sbb-textexpand

Properties
Name Description
@Output()

expandEvent: EventEmitter<boolean>

Event activated at the expansion of the text.

ariaLive: 'polite'

Is used to set the priority with which screen reader should treat updates to live regions. With "polite" value the screen reader will speak changes whenever the user is idle.

collapsedComponent: TextexpandCollapsedComponent

Refers to the textexpand-collapsed component istance.

cssClass: true

Css class of the textexpand component.

expandedComponent: TextexpandExpandedComponent

Refers to the textexpand-expanded component istance.

id: `sbb-textexpand-${counter++}`

Identifier of the textexpand component.

isExpanded: false

Describes if text content is expanded or not. Initially is collapsed.

role: 'region'

Is an ARIA landmark role. It provide a way to identify a specific zone of the page. Screen readers use landmark roles to provide keyboard navigation to important sections of a page.
Methods
toggleExpanded

TextexpandExpandedComponent

Properties
Name Description
