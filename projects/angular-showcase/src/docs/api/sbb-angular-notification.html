API reference for Angular SACL notification

import { NotificationsModule } from '@sbb-esta/angular-public';

Directives

NotificationComponent

Selector: sbb-notification

Properties
Name Description
@Input()

icon: TemplateRef<any>

The icon to be used into the notification left side. By default uses three icons for SUCCESS, ERROR or INFO notification type, but the user can use his own icon using the NotificationIconDirective.
@Input()

jumpMarks: JumpMark[]

List of in page links displayed on the bottom of the notification
@Input()

message: string

Message displayed into the notification content
@Input()

type: 'success' | 'info' | 'error'

Type of notification.
Methods
scrollTo

Used to scroll to an element identified by a jump mark
Parameters

$event

any

click event

JumpMark

jump mark after the notification message
