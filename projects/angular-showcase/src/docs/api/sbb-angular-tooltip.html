API reference for Angular SACL tooltip

import { TooltipModule } from '@sbb-esta/angular-public';

Directives

TooltipComponent

Selector: sbb-tooltip

Properties
Name Description
@Input()

icon: TemplateRef<any>

The icon to be used as click target. By default uses question-mark, but the user can use his own icon using the TooltipIconDirective.
@Output()

closed: EventEmitter<SbbTooltipChangeEvent>

Close event to a click on tooltip element.
@Output()

opened: EventEmitter<SbbTooltipChangeEvent>

Open event to a click on tooltip element.

contentId: 'sbb-tooltip-content-id-' + tooltipCounter++

Identifier of tooltip content.

cssClass: true

Css class on tooltip component.

overlayAttached:

Checks if a tooltip panel exists

panelClosingActions: Observable<SbbTooltipChangeEvent | null>

A stream of actions that should close the autocomplete panel, including on blur events.

tooltipId: 'sbb-tooltip-id-' + tooltipCounter++

Identifier of tooltip.

tooltipRef: OverlayRef

Overlay containg the tooltip text and the close button. It's built when the user clicks on the question mark.

tooltipTrigger: ElementRef<any>

Refers to tooltip trigger element.
