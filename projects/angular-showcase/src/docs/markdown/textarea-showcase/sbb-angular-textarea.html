TextArea Overview

Import textarea module into your application

import { TextareaModule } from '@sbb-esta/angular-public';

and then you can use the textarea component as seen below

<h4 class="sbbsc-block">Textarea</h4>
<sbb-textarea
  [(ngModel)]="textArea1"
  [minlength]="minlength"
  [maxlength]="maxlength"
>
</sbb-textarea>
